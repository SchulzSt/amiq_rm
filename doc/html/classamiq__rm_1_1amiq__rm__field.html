Detailed Description

This class is used to model a field (which can be contained in a register). The field is defined through a name, reset_value, size and attribute. The behavior of the attribute is defined through virtual functions like: is_readable(), is_writable(), etc. To add additional field attributes, the virtual functions mentioned earlier should be overwritten accordingly.

std::string amiq_rm::amiq_rm_field::atrib

The attribute of the field represents the behavior of the field at access. The implemented attributes:
"RO" - W: no effect, R: no effect
"RW" - W: as-is, R: no effect
"RC" - W: no effect, R: clears all bits
"WO" - W: as-is, R: error
"RS" - W: no effect, R: sets all bits
"WRC" - W: as-is, R: clears all bits
"WRS" - W: as-is, R: sets all bits
"WC" - W: clears all bits, R: no effect
"WS" - W: sets all bits, R: no effect
"WSRC" - W: sets all bits, R: clears all bits
"WCRS" - W: clears all bits, R: sets all bits
"W1C" - W: 1/0 clears/no effect on matching bit, R: no effect
There is a mechanism which relies on functions like is_writable(), is_readable(), etc. through which the attribute's behavior is extracted and used by the register at access.
